CSCE 242: Jackson Devlin

CSCE 242 is a class intended to show proper format and design of websites using HTML and CSS through exercises, assignments, and a semester long project.

GitHub Repository

Assignments

  • Assignment 1: Basic HTML

    • Assignment provides fundamental formatting and understanding of HTML.

  • Assignment 2: Basic CSS

    • Provides proper formatting and understanding of CSS coding building off the HTML fundamentals giving style to what was previously learned.

  • Assignment 3: Flex Box

    • Building off what was learned for both CSS and HTML, it pushes website flair further by flexing the website to align with a smaller or larger screen.

  • Assignment 4: CSS Recreation

    • Builds more on CSS and website formation while introducing new ways of formatting to make websites look more appealing.

  • Assignment 5: JavaScript Buttons, Functions, and More

    • Introduction to JavaScript for website design and show introductory basics in its usage

  • Assignment 6

    • Building on JavaScript fundamentals, this assignment introduces conditional statements and their usage in programming.

  • Assignment 7: Loops

    • Building more on JavaScript, this assignment introduces loops and their usage in programming.

  • Assignment 8: Arrays

    • Building more on JavaScript, this assignment introduces arrays and their usage in programming.

  • Assignment 9: Classes

    • Building more on JavaScript, this assignment introduces classes and their usage in programming.

Project

  • Project Assignment 1: Topic Selection

    • The first part of the project is to select a topic and create a proposal for the website that will be built.

  • Project Assignment 2: Wireframes

    • The second part of the project is to design wireframes to outline the structure and layout of the website before development.

  • Project Assignment 3: Project HTML and CSS

    • The First start on the project to layout the html and css of the website making it similar to wireframe

  • Project Assignment 4: Project Pictures, Color, and Text

    • working on the inital HTML and CSS and actually loading data and content into the website

  • Project Assignment 5: Project Brushups

    • taking the comments and remarks from teachers and implementing into the website making it better

  • Project Assignment 6 & 7: Json Parsing and Contact Form

    • learning how to use Json for data entry and parsing it so it appears on website and using contact forms for users to reach out to website owner.

  • Project Assignment 8: React
    • Code Repository
    • React Static Code
    • Backend Code Repo

    learning how to use React for building user interfaces and managing state in web applications.
